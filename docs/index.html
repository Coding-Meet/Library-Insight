Why Library Insight?

Designed for developers, documentation generators, and AI assistants.

📦

Bytecode Inspection

Parses compiled Java class signatures and decodes Kotlin metadata to retrieve native declarations.

🔍

Sources & Javadoc Matching

Matches class definitions to source files automatically to attach documentation comments and method bodies.

🌐

On-the-Fly Maven Resolution

Resolves Maven coordinates automatically over HTTP using Ktor client, caching targets locally.

🤖

Split AI Context

Generates compact, split-file package and class directories under build/ai-context/ optimized for LLM token usage.

Command Line Reference

Explore CLI command options and interactive usage guides.

Scan Library & Maven Coordinates

Analyze local files or download from repositories directly to populate the indexing database.

# Scan Retrofit from Maven Central (jar + sources)
library-insight scan com.squareup.retrofit2:retrofit:2.11.0

# Scan OkHttp client
library-insight scan com.squareup.okhttp3:okhttp:4.12.0

# Scan Aldebaran QiSDK library
library-insight scan com.aldebaran:qisdk:1.7.5

Search Index Symbols

Quickly search for packages, classes, methods, or property declarations matching a search term.

# Search matching classes or methods
library-insight search Retrofit
library-insight search OkHttpClient

# Search for animation view classes or db descriptors
library-insight search LottieAnimationView

Explain Class Details

Generate detailed structural reports displaying constructors, signatures, and Javadoc definitions.

# Inspect structural details of Retrofit class
library-insight explain Retrofit

# Inspect structural details of OkHttpClient
library-insight explain OkHttpClient

Export and Diffing

Save reports to clean Markdown sheets, JSON documents, or perform a version comparison to detect breaking changes.

# Export index to Markdown
library-insight export markdown > API_REFERENCE.md

# Diff two archive versions to detect breaking changes
library-insight diff retrofit-2.9.0.jar retrofit-2.11.0.jar

AI Context Generator

Export a token-efficient split directory layout mapping package namespaces to folders and classes to individual files.

# Generates output under build/ai-context/
library-insight ai-export

Agent Skills Management

Initialize or manage custom AI agent skills for the current workspace (.agents/skills/) directly from the CLI.

# Initialize workspace skill
library-insight init

# Install/update skill to workspace
library-insight skills add

# Remove workspace skill
library-insight skills remove

# List installed skills
library-insight skills list

Get Started Globally

Install and link the CLI executable globally on your computer in seconds.

Option A: Install globally via npm (Recommended):

npm install -g library-insight

Option B: Run from the repository via installer script:

chmod +x install.sh
./install.sh

Once finished, the library-insight command will be globally configured on your terminal path.
